# application-of-data-science
本リポジトリでは，九州大学工学部機械工学科の専攻教育科目「データサイエンス応用」の課題で提出した自身の解答をまとめています．

## 使用技術
* Python
* Jupyter Notebook
* Numpy
* pandas
* Matplotlib
* scikit-learn

## 学びと今後の展望
本講義を通して，線形回帰，決定木，Random Forest, k-means法などの機械学習法を適切に選択し，実行・評価するための基本的知識を身につけることができました．
